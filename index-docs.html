⚡ Quick Start (5 Minutes)

  1. Open phpMyAdmin: http://localhost/phpmyadmin
  2. Create Database: Create new database named plum_db
  3. Import Schema: Go to Import tab, select database/schema.sql
  4. Visit Homepage: http://localhost/plumart/index.html
  5. Test API: http://localhost/plumart/api/artists
